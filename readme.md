# ML Git Lab – House Price Prediction

## 📌 Project Overview
This project demonstrates a Machine Learning workflow for predicting house prices.  
It is used to practice Git version control and experiment with ML models.

## 📂 Project Files

- dataset1.csv → raw housing dataset  
- dataset2.csv → processed/clean dataset  
- model.ipynb → Jupyter Notebook containing data preprocessing, training and evaluation  
- README.md → project documentation  

## ⚙️ Tasks Performed
✔ Data preprocessing  
✔ Feature engineering  
✔ Model training  
✔ Model evaluation  

## 🤖 Model Used

- Decision Tree (experimental model)
